Big Game Survey

Big Game Survey é uma aplicação full stack web e mobile construída durante a 1ª edição da Semana DevSuperior. A aplicação consiste em uma pesquisa de preferência de games, onde os dados são coletados no app mobile, e depois são listados no app web, que também apresenta um dashboard com gráficos baseados nestes dados.Foram utilizadas as tecnologias ReactJS com Typescript no Front-end e Java com SpringBoot no Back-end e Banco de dados Postgres. Implantado no Heroku e Netlify.

DSMeta

DSMeta é um projeto realizado no evento da DevSuperior. Foram utilizadas as tecnologias ReactJS com Typescript no Front-end e Java com SpringBoot no Back-end e Banco de dados Postgres. Implantado no Heroku e Netlify.

Widget Feedback

About Widget Feedback é uma aplicação full stack web e mobile construída durante a NLW return, trilha impulse evento organizado pela Rocketseat. A aplicação consiste em um Widget no qual é possível relatar um problema e dar sugestões assim os dados são coletados no server, e depois são enviados para o Mailtrap.
